# Jaffx Audio Inc.

**Affordable, portable, hackable digital audio for everybody.**

Jaffx Audio Inc. is dedicated to making great digital audio tools accessible through open-source software and affordable hardware. We develop tools and libraries that empower musicians, sound designers, and audio developers to create and experiment to their hearts' content.

## Featured Projects

### [JAFFX](https://github.com/jaffco/JAFFX)
A free, open-source platform for programming the JAFFX reprogrammable pedal and other Daisy Seed-based projects. Built around a single-header library that simplifies audio DSP development for embedded hardware.

### [Gimmel](https://github.com/jaffco/Gimmel)
A lightweight, cross-platform C++ audio effects library designed with human-readability and pedagogy in mind. Gimmel provides well-documented implementations of classic digital audio effects, making it ideal for learning and development.
